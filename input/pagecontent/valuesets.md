### Value Sets

This implementation guide defines the following value sets:

- [Tooth Identification FDI](ValueSet-tooth-identification-fdi.html) — FDI/ISO 3950 tooth numbering
- [Dental Finding Codes](ValueSet-dental-finding-codes.html) — LOINC/SNOMED codes for dental findings
- [Dental Procedure Codes](ValueSet-dental-procedure-codes.html) — BEMA + GOZ + OPS procedure codes
- [Dental Befundstatus](ValueSet-dental-befund-status-vs.html) — KZBV tooth status codes
- [BEMA Codes](ValueSet-bema-codes-vs.html) — All BEMA billing codes
- [GOZ Codes](ValueSet-goz-codes-vs.html) — All GOZ billing codes
